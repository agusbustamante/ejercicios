Índice básico — especiales_Php

Enlaces simples a las carpetas del proyecto. Abre esto en tu servidor local (XAMPP) en:

http://localhost/ejercicios/Php/especiales_Php/

  • Base de datos: lista ordena y filtra

    • Si el servidor muestra aún otro índice, asegúrate de que Apache está ejecutándose y que index.html está en la raíz de la carpeta especiales_Php. Apache suele servir index.html antes que index.php; si ves el PHP en lugar del HTML, puedes borrar o renombrar temporalmente index.php.
