Ajuste automático de fuentes en diferentes pantallas

En este ejemplo, los títulos son más grandes que el texto común, para destacar la información principal.

Se establece un tamaño base para todo el documento y se emplean unidades relativas para mantener proporciones.

Con la ayuda de media queries, la tipografía se ajusta automáticamente según el ancho de la ventana.

La resolución de pantalla se mide en píxeles de ancho por píxeles de alto, y cada dispositivo puede variar.

Equipos pequeños, como teléfonos modernos, suelen tener gran densidad de píxeles, lo que permite ver texto más nítido.

Por otro lado, pantallas grandes tienden a tener menos densidad, mostrando elementos más grandes.

En este caso, se usan tres rangos para simular distintos dispositivos.

Si la densidad es alta, la percepción es de alta calidad visual; si es baja, el contenido parece más grande.

Se prioriza ajustar el texto para mantener la legibilidad en todas las resoluciones.

Para esta práctica:

— 14px para pantallas pequeñas (móviles y tablets en vertical)

— 22px para pantallas medianas (entre 800px y 1599px de ancho)

— 32px para pantallas grandes (a partir de 1600px)
