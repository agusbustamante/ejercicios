Encolumnado sin Tablas

Columna 1

Las media queries son fundamentales en el diseño responsivo. Permiten aplicar estilos CSS específicos según las características del dispositivo, como el ancho de pantalla. En este ejemplo se utilizan breakpoints en 1200px, 900px y 600px para crear diferentes layouts según el tamaño de la pantalla del usuario.

@media screen and (max-width: 600px)

Columna 2

Este diseño utiliza la propiedad float: left para crear columnas. Aunque es una técnica más antigua, sigue siendo efectiva para layouts simples. Cada columna tiene un ancho calculado dinámicamente usando calc() que resta los márgenes del porcentaje total, permitiendo diseños flexibles y responsivos.

float: left; width: calc(20% - 20px)

Columna 3

El box model es crucial para entender cómo se calculan las dimensiones. Aquí se usa box-sizing: border-box para que el padding y border se incluyan en el ancho total, facilitando los cálculos responsivos.

Columna 4

Los breakpoints definen cuándo cambia el layout. En pantallas grandes (>1200px) se muestran 5 columnas, en medianas (901-1200px) 3 columnas, en pequeñas (601-900px) 2 columnas, y en móviles (<600px) una sola columna apilada verticalmente.

Columna 5

Las imágenes se adaptan automáticamente con max-width: 100% y height: auto. Esto asegura que nunca excedan el ancho de su contenedor y mantengan su proporción original.

max-width: 100%; height: auto;

Columna 6

Se utilizan Google Fonts para mejorar la tipografía. Playfair Display para títulos elegantes y Source Sans Pro para el texto del cuerpo, combinando serif y sans-serif para crear jerarquía visual efectiva.

Columna 7

Los box-shadow agregan profundidad y dimensión al diseño. Text-shadow mejora la legibilidad de los títulos. Estos efectos CSS crean una experiencia visual más rica sin necesidad de imágenes adicionales, manteniendo el rendimiento óptimo.

Columna 8

Los colores y contrastes son fundamentales para la usabilidad. En este diseño se utilizan tonos cálidos como naranja y chocolate que crean una paleta armoniosa. El contraste adecuado entre texto y fondo asegura que el contenido sea legible en todos los dispositivos.
