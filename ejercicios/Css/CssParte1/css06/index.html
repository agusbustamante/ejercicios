Ejemplos de Centrado y Padding

Esta es la primera caja de demostración
Aquí puedes ver cómo se comporta el texto
sin ningún tipo de padding aplicado
El contenido llega hasta los bordes
de manera natural
(no padding)

Esta es la segunda caja demostrativa
Se observa cómo el padding horizontal
crea espacio interno en los lados
El texto se separa de los bordes
mejorando la legibilidad
(padding-right: 150px; padding-left: 150px; no box-sizing)

Esta es la tercera caja de ejemplo
Aquí se demuestra el uso de box-sizing
Con border-box el padding se incluye
dentro del ancho total especificado
Manteniendo las dimensiones controladas
(padding-right: 150px; padding-left: 150px; box-sizing: border-box)
